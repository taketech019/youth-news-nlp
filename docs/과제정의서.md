## Part 1. NLP 관련 핵심 개념어 정리

### 1. 데이터 관점

**빅데이터 (Big Data)**
3V를 핵심 속성으로 한다 — Volume(대용량), Velocity(생성 속도), Variety(다양한 형태). 단순히 데이터의 양적 측면만이 아니라 데이터의 수집·저장·분석·시각화 등 전 과정을 포괄하는 개념이다 (Laney, 2001; 함유근·채승병, 2012).

**비정형 데이터 (Unstructured Data)**
관계형 데이터베이스의 정형화된 열/행 구조에 맞지 않는 텍스트·이미지·영상 등의 데이터. 신문 기사 본문은 대표적인 비정형 텍스트 데이터이며, 형태소 분석 등의 가공 과정을 거쳐야 분석 가능한 형태가 된다.

**소셜 빅데이터 (Social Big Data)**
소셜 미디어·뉴스 등 온라인에서 생성되는 대용량 텍스트 데이터. "그 시대의 감성과 정서를 포함하는 원천"으로서 공공정책 여론 탐색에 활용될 수 있다. 사람이 직접 코딩하지 않아 코더 간 신뢰도 문제로부터 자유롭고, 전집(全集)에 가까운 분석이 가능해 표본오차가 없다는 장점이 있다 (박대민, 2013).

### 2. 텍스트 전처리 (Text Preprocessing)

**형태소 분석 (Morphological Analysis)**
한국어는 어간에 조사·어미·접사가 결합되어 복합어를 이루는 교착어이므로, 의미 단위를 추출하려면 단어를 형태소로 분해한 뒤 접두어·접미어·조사 등을 제거해야 한다 (김용학·김영진, 2016). 논문에서는 Textom을 사용했으며, 본 연구에서는 **kiwi(kiwipiepy)**를 사용한다.

**동의어 통일 (Synonym Normalization)**
동일한 의미를 가진 서로 다른 표현을 하나의 대표어로 통일하는 작업. 예: "애기/애/아기" → "아이", "출산 장려금/출산장려금" → "출산장려금". 이 작업을 거치지 않으면 의미연결망에서 특정 단어의 빈도가 과도하게 분산되어 중요 노드를 식별하기 어려워진다.

**단어×단어 행렬 (Co-occurrence Matrix)**
유사 맥락에서 공동으로 출현하는 단어 간의 빈도를 담은 대칭 행렬. 한 문장 혹은 문단을 분석 단위로 하여 두 단어가 동시에 나타난 횟수를 셈한다. 의미연결망 분석의 입력 데이터가 되며, 맥락 내에서의 단어 간 관계를 수치화한 결과물이다 (차민경, 2015).

### 3. 네트워크 기본 속성

**노드(node, vertex)와 링크(link, edge)**
소셜 네트워크에서 노드는 개별 행위자를, 링크는 행위자 간의 관계·연결·교환을 나타낸다. 의미연결망에서는 **노드 = 단어(키워드)**, **링크 = 단어 간 공출현 관계**로 치환된다.

**방향성 (Directionality)**

- **무방향 네트워크 (Undirected Network):** A-B와 B-A가 동일한 관계 (예: 논문 공저자 네트워크)
- **방향 네트워크 (Directed Network):** A→B와 B→A가 서로 다른 관계 (예: 트위터 팔로잉, 리트윗)

의미연결망 분석은 일반적으로 단어 간 공출현에 기반하므로 **무방향 네트워크**로 구성된다.

**네트워크 크기·밀도·포괄성**

- **네트워크 크기 (network size):** 노드의 총 수
- **네트워크 밀도 (network density):** 실제 연결된 링크 수 / 이론적 최대 링크 수
    - 무방향 이진 네트워크: k / {n×(n-1)/2}
    - 방향 이진 네트워크: k / {n×(n-1)}
- **포괄성 (inclusiveness):** 네트워크 전체 노드 중 고립되지 않은 노드의 비율 = (n − n_isolated) / n

**연결정도 (degree)와 연결거리 (distance)**

- **연결정도:** 한 노드에 직접 연결된 링크 수. 최대 n−1까지 가능하며, 고립 노드는 0.
- **연결거리 (shortest path distance):** 두 노드를 잇는 가장 적은 수의 링크로 된 경로. 네트워크 내 정보 전달 속도의 지표.
- **평균 연결거리 (average distance):** 임의의 두 노드가 평균적으로 몇 단계로 연결되는지 보여주는 지표.

### 4. 중심성 분석 (Centrality Analysis)

네트워크 내에서 특정 노드가 **얼마나 중심적인 역할을 수행하는지** 측정하는 지표군. 의미연결망에서는 어떤 단어가 담론의 핵심인지를 판단하는 근거가 된다.

| 지표 | 정의 | 의미 |
| --- | --- | --- |
| **연결정도 중심성 (Degree Centrality)** | 한 노드에 직접 연결된 노드 수 (Freeman, 1979) | 해당 단어가 다른 단어들과 얼마나 폭넓게 공출현하는가 — 활동성·영향력 |
| **매개 중심성 (Betweenness Centrality)** | 한 노드가 다른 노드 쌍의 최단 경로상에 위치하는 횟수 (Bonacich, 1972) | 서로 직접 연결되지 않은 단어들 사이에서 **중개자(gate keeper)** 역할. 정보 흐름을 좌우 |
| **아이겐벡터 중심성 (Eigenvector Centrality)** | 연결된 이웃 노드의 중요도를 가중치로 반영 (Bonacich, 2007) | "중요한 단어와 연결된 단어"를 포착. 소수의 영향력 높은 노드와 연결될수록 증가 |

연결정도 중심성만 높은 것은 네트워크에서 중심적이라고 단정할 수 없다 — 고립된 노드 여러 개와 연결된 노드 A보다, 영향력 있는 노드 몇 개와 연결된 노드 B가 더 중심적일 수 있기 때문이다 (최수진, 2016).

**네트워크 중심도 (Network Centralization)**
개별 노드가 아니라 **네트워크 전체의 중심화 경향**을 나타내는 지표. 소수의 단어에 연결이 집중되면 중심도가 높고, 여러 단어가 분산적으로 연결되어 있으면 중심도가 낮다 (정덕호·유대영, 2013).

### 5. 군집 분석 (Clustering)

**콘커 분석 (CONCOR; CONvergent CORrelation)**
상관관계 행렬을 반복적으로 수렴시켜 블록(군집)을 생성하는 군집 분석 기법. **구조적 등위성(structural equivalence)** 분석의 일종으로, 연결망 내에서 서로 같은 유형의 관계 패턴을 보이는 노드를 묶어낸다 (Burt, 1978; 김용학·김영진, 2016).

직접 연결된 노드 간 유사성은 일반 커뮤니티 탐지로 파악할 수 있지만, **간접적으로 연결된 노드 간 유사성**을 판별할 때 CONCOR가 유용하다. 참고 논문에서는 110개 단어를 8개 그룹(비혼 의지, 주변 환경, 만혼 인식, 가치관 변화, 미디어 영향, 변화 상징, 임신·육아 걱정, 국가적 노력)으로 구분하는 데 활용되었다.

**커뮤니티 탐지 (Community Detection)의 대안**
Python/Gephi 환경에서 CONCOR를 직접 대체할 수 있는 알고리즘으로는 **Modularity 기반의 Louvain 알고리즘**, **Girvan-Newman 알고리즘**, **Spectral Clustering** 등이 있다. Gephi는 기본적으로 Louvain을 지원한다.

### 2.3 단계별 상세 계획

### Step 1. 데이터 전처리

참고 논문이 "저출산과 관련 없는 연예인 출산·동물 출산·제품 후기"를 제거한 것처럼, 본 연구도 "청년" 키워드가 **사회·문화적 청년 담론과 무관한 맥락**에서 등장한 기사를 제거한다.

- **중복 제거:** 동일 기사의 재전송 버전
- **노이즈 제거 예시:** "청년 아이유", "청년 손흥민" 등 인명 수식어로 쓰인 기사 / 부고 기사 / "청년회관", "청년상회" 등 고유명사 위주 기사 / 광고성 기사
- **메타데이터 보존:** 발행일자(연·월·일), 언론사, 기사 ID — RQ3 시계열 분석에 필수

### Step 2. 형태소 분석 (kiwi)

```python
from kiwipiepy import Kiwi
kiwi = Kiwi()

# 도메인 특화 명사 사전 등록 (중요)
kiwi.add_user_word("N포세대", "NNP")
kiwi.add_user_word("이대남", "NNP")
kiwi.add_user_word("이대녀", "NNP")
kiwi.add_user_word("청년기본법", "NNP")
# ... 청년 담론의 신조어·복합명사를 사전에 추가

def extract_nouns(text):
    tokens = kiwi.tokenize(text)
    # NNG(일반명사), NNP(고유명사)만 추출
    return [t.form for t in tokens if t.tag in ("NNG", "NNP") and len(t.form) > 1]
```

- **품사 제한:** 의미연결망 분석은 **명사 중심**(NNG, NNP)이 관례. 참고 논문도 "명사 위주로 동의어를 통일"한다고 명시.
- **한 글자 명사 제거:** "돈", "집" 등은 의미는 있으나 형태소 분석 오류(조사 잔류 등)로 혼입되는 경우가 많으므로 사전 검토 후 결정.

### Step 4. 공출현 행렬 구축

분석 단위를 **문장**으로 할지 **문단**으로 할지, 아니면 **기사 전체**로 할지 결정해야 한다. 참고 논문은 트윗 단위(짧은 텍스트)였지만, 일간지 기사는 길이가 훨씬 길다.

- **권장:** **문장 단위** 공출현 — 의미적 밀착도가 높음
- **대안:** 기사 단위 — 네트워크 밀도가 매우 높아짐(과밀화 위험)

```python
from collections import Counter
from itertools import combinations
import pandas as pd

cooc = Counter()
for sentence_tokens in sentences:
    unique_tokens = sorted(set(sentence_tokens))
    for w1, w2 in combinations(unique_tokens, 2):
        cooc[(w1, w2)] += 1
```

### Step 6. 의미연결망 분석 (RQ2 해결)

**(1) 네트워크 전체 지표 (Python + networkx)**


### 2.5 기대 기여

1. **방법론적 기여:** 참고 논문이 트위터 단일 시점 분석이었다면, 본 연구는 **일간지 + 시계열**로 분석 축을 확장한다. 특히 한국어 형태소 분석기(kiwi)와 Python–Gephi 오픈 스택으로 수행하여 **재현 가능성**을 높인다.
2. **이론적 기여:** 청년 담론이 어떤 의미적 하위 구조로 이루어져 있으며 지난 수년간 어떻게 재편되었는지를 실증적으로 규명한다.
3. **정책적 함의:** 매개 중심성이 높은 "교량 단어"의 변화는 청년 정책이 접근해야 할 개념적 결절점(nodal point)을 시사한다.
